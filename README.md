# 🟫 Lenzcraft – A Minecraft-Inspired Sandbox Game

Lenzcraft is a Minecraft-style sandbox game built in Unity. It features procedurally generated terrain, block placement/removal, basic crafting mechanics, and a first-person control system. The project is designed as a learning exercise in game development and voxel-based worlds and is still in development.

---

## 🎮 Features

- 🧱 **Block-Based World**: Chunk-based terrain system with infinite horizontal generation.
- 🌍 **Procedural Generation**: Uses Perlin noise for realistic-looking terrain.
- 🪓 **Block Interaction**: Place and remove blocks in real time with mouse clicks.
- 👀 **First-Person Controller**: Move, jump, and look around freely in a 3D world.
- 💡 **Lighting and Shadows**: Basic day-night cycle with dynamic lighting.
- 📦 **Inventory System** *(optional)*: Hotbar system for selecting and managing blocks.
- 🔧 **Optimized Chunk Loading**: Only nearby chunks are loaded/rendered to boost performance.

---

## 🧰 Tech Stack

- **Engine**: Unity (2022 or later)
- **Language**: C#
- **Tools**: Unity's Input System, Custom Mesh Generation, Perlin Noise, Scriptable Objects
